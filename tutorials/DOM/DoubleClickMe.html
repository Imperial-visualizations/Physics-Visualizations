HTML DOM: CSS, Javascript & JQuery

Getting Started

How to get started:
1. Have your files form the HTML tutorial ready, or download them from /Imperial-Visualizations/tutorials/DOM/
2. Open "index.html" with PyCharm or whatever code-editor you like:
3. If it's from /tutorials/HTML, you need to add one extra line:

<script src="https://code.jquery.com/jquery-3.2.1.min.js"></script>
before the line 
<script src="scripts/program.js"></script>
, which both should be right before the </body> tag.
4. Start adding Javascript/JQuery codes between the <script></script> tags.

index.html
<!doctype html>
<!-- **This line is extremely important, it tells the browser to use HTML5 instead of HTML 4.1**  -->
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
    <title>Imperial Visualisations</title>
    <link rel="icon" href="images/favicon.ico" type="image/x-icon">
    <link rel="stylesheet" href="styles/styles.css">
    <!-- importing style.css -->
    <!-- **You are not suppose to change style.css !!**  -->
    <style type="text/css">
        /* overwrite style.css styles*/
    </style>
</head>
<body>
    <div class="container">
        <h1>Big title</h1>
        <div class="row">
            <div class="eight columns">
                <p>Put your graph here</p>
            </div>
            <div class="four columns">
                <p>And probobly put your interface here</p>
            </div>
        </div>
    </div>
    <br><br><br>
    <script src="scripts/program.js"></script>
    <!-- You should put your javascript code in program.js -->
    <script>
        // or you can put your javascript here as well.
    </script>
</body>
</html>
